A few options:

## 1. Claude Code Hooks (Best approach)

Set up a `PostToolUse` or `Stop` hook that automatically saves transcripts. Add to your Claude Code settings (`~/.claude/settings.json`):

```json
{
  "hooks": {
    "Stop": [
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "echo \"$CLAUDE_TRANSCRIPT\" >> ~/projects/media-app/transcripts/$(date +%Y%m%d_%H%M%S).md"
          }
        ]
      }
    ]
  }
}
```
